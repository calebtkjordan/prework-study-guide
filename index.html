CSS

  • CSS is used to style and layout webpages
  • CSS can be included as external files, internal styles within styletags, or inline within HTML elements.
  • The box model in CSS consists of margins, borders, padding, and content for element layout.
  • Responsive design in CSS enables webpages to adapt to different screen sizes using media queries.
  • CSS supports animations and transitions for dynamic effects on webpage elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add . : stages all changes in the current directory
  • git branch list your branches. a * will appear next to the currently active branch
  • git checkout switch to another branch and check it out into your working directory

JavaScript

  • JavaScript is a programming language used to create interactive effects within web browsers.
  • It supports event-driven, functional, and imperative programming styles.
  • JavaScript can manipulate HTML content and CSS styles, enhancing user interaction.
  • It's an integral part of web development, alongside HTML and CSS.
  • JavaScript supports asynchronous operations using callbacks, promises, and async/await.
  • Modern JavaScript has frameworks and libraries like React, Angular, and Vue.js for robust web development.
  • JavaScript runs in most modern web browsers without needing any additional plugins.
